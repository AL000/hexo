主题开发

Last updated: Feb 20, 2013

结构

|-- _config.yml
|-- languages
|-- layout
|-- source

_config.yml

主题的设定文件。

参考：设定

languages

主题的语言文件。

参考：国际化（i18n）

layout

模板文件夹。你可使用 EJS 或 Swig 处理模版文件，或安装 Renderer 插件 以使用你喜爱的样板引擎。

基础布局：

  • archive - 存档列表，若无此模板则使用 index 布局。
  • category - 分类，若无此模板则使用 archive 布局。
  • index - 首页
  • page - 分页，若无此模板则使用 index 布局。
  • post - 文章，若无此模板则使用 index 布局。
  • tag - 标签，若无此模板则使用 archive 布局。

你也可以在主题中自定其他布局，例如link或photo之类的，若找不到自定的布局的话，则会根据 Generator 的不同，使用相对应的布局代替。

（至少要有index布局）

source

源文件，CSS、JavaScript 等文件（Asset）应放置于此文件夹。此文件夹内的文件在经过处理后会被复制至public文件夹，文件或文件夹名称开头为.（点）或_（底线）的会被忽略。

Hexo 内建 Stylus 及 nib 支持，你可安装 Renderer 插件 让 Hexo 支持更多文件格式。

参考

你可参考预设主题 Light 来制作主题。
