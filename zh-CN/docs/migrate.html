迁移

Last updated: Feb 20, 2013

目录

  • RSS
  • Jekyll / Octopress
  • WordPress

RSS

首先，执行以下指令安装 RSS Migrator 插件，并在_config.yml中的plugins栏位加入hexo-migrator-rss。

npm install hexo-migrator-rss

安装完毕后，执行以下指令即可，source参数可为文件路径或网址。

hexo migrate rss <source>

Jekyll / Octopress

将source/_posts内的档桉拷贝至 Hexo 的source/_posts内，并将文章内的categories属性修改为tags。

然后调整 _config.yml 中的 new_post_name 为 :year-:month-:day-:title.md。

WordPress

首先，执行以下指令安装 WordPress Migrator 插件，并在_config.yml中的plugins栏位加入hexo-migrator-wordpress。

npm install hexo-migrator-wordpress

安装完毕后，执行以下指令即可，source参数可为文件路径或网址。

hexo migrate wordpress <source>
