安装

Last updated: Feb 20, 2013

在安装 Hexo 之前，必须先安装 Git 和 Node.js。

目录

  • Windows
  • Mac
  • Linux

Windows

Git

下载 msysgit 并执行即可完成安装。

Node.js

在 Windows 环境下安装 Node.js 非常简单，仅须 下载 安装文件并执行即可完成安装。

Hexo

利用 npm 即可安装。

npm install -g hexo

Mac

Git

Mac 已内建 Git，虽然版本较旧，不过还是能用，如果你想要使用较新版的 Git 的话，可参考以下方式安装。

  1. 使用 Homebrew。

     brew install git

  2. 使用 MacPorts。

     sudo port install git-core

  3. 使用 安装程式。

如果出现错误情况，可能是尚未安装编译工具，请至 App Store 下载 Xcode。

Node.js

Mac 环境下有很多安装方式可供选择，以下使用 nvm 来安装。

执行以下指令。

git clone git://github.com/creationix/nvm.git ~/nvm

安装完毕后，在~/.bash_profile或~/.bashrc加入以下内容并重开一个新的终端视窗。

. ~/nvm/nvm.sh

之后即可使用 nvm 来安装 Node.js 了，编译可能需要花些时间，请耐心等候。

nvm install v0.8.14
nvm use v0.8.14
nvm alias default v0.8.14

Hexo

利用 npm 即可安装。

npm install -g hexo

Linux

Git

Ubuntu, Debian:

sudo apt-get install git-core

Fedora, Red Hat, CentOS:

sudo yum install git-core
