服务器

Last updated: Feb 20, 2013

内建服务器

Hexo 使用 Connect 作为静态文件的服务器。

编辑_config.yml中的port栏位调整服务器的连接端口。

port: 4000

执行以下命令启动服务器，按下Ctrl+C关闭服务器。加入 -p/--port 选项设定连接端口。

hexo server
hexo server -p 12345

记录器

编辑_config.yml中的logger栏位启动记录器。编辑logger_format栏位可调整记录的显示内容，参考 Connect 以获得更多资讯。

logger: true
logger_format:

Pow

Pow 是由 Node.js 所建立的 Mac 环境专用的零配置 Rack 服务器，不过它也能用于处理一般的静态文件。

安装

执行以下命令即可完成安装。

curl get.pow.cx | sh

使用

在~/.pow建立链接即可使用。

cd ~/.pow
ln -s /path/to/myapp

完成后，网站即会出现在http://myapp.dev，网址根据链接名称而有所不同。

参考 Pow 以获得更多资讯。
