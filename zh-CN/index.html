快速、简单且功能强大的 Node.js 博客框架。
Tweet

功能

  • 不可思议的快速 ─ 只要一眨眼静态文件即生成完成
  • 支持 Markdown
  • 仅需一道指令即可部署到 GitHub Pages 和 Heroku
  • 已移植 Octopress 插件
  • 高扩展性、自订性
  • 兼容于 Windows, Mac & Linux

安装

npm install -g hexo

更新

npm update -g

开始

建立项目：

hexo init project
cd project

建立新文章：

hexo new "New Post"

生成静态文件：

hexo generate

启动服务器：

hexo server
