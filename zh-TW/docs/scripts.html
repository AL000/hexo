腳本

Last updated: Mar 1, 2013

腳本（Script）相較於外掛（Plugin），是規模較小且僅有單一檔案的擴充元件。

使用

將 JavaScript 檔案放置於scripts資料夾，即會自動載入，檔案或資料夾名稱開頭為_（點）或.（底線）的會被忽略。

範例

文章建立後，自動開啟檔案。

var spawn = require('child_process').spawn;

hexo.on('new', function(){
  spawn('open', [target]);
});
