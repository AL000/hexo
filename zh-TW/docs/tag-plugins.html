標籤外掛

Last updated: Jan 6, 2013

標籤外掛（Tag Plugin）與文章中的標籤（Tag）不同，標籤外掛用於在文章內快速加入特定內容。

Block Quote

插入 Block Quote。

{% blockquote [author[, source]] [link] [source_link_title] %}
content
{% endblockquote %}

別名：quote

參考

Code Block

插入程式碼區塊。

{% codeblock [title] [lang:language] [url] [link text] %}
content
{% endcodeblock %}

別名：code

參考

Backtick Code Block

插入程式碼區塊。

``` [language] [title] [url] [link text]
content
```

參考

Pull Quote

插入 Pull Quote。

{% pullquote [class_name] %}
content
{% endpullquote %}

參考

jsFiddle

插入 jsFiddle。

{% jsfiddle shorttag [tabs] [skin] [width] [height] %}

參考

Gist

插入 Gist。

{% gist gist_id [filename] %}

參考

Image Tag

插入圖片。

{% img [class names] /path/to/image [width] [height] [title text [alt text]] %}

參考

Youtube

插入 Youtube 影片。

{% youtube video_id %}

Vimeo

插入 Vimeo 影片。

{% vimeo video_id %}

Raw

逸脫處理。（請去掉反斜線）

{% raw %}
{% endraw /%}
