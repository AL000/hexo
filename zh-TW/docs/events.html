事件

Last updated: Jan 12, 2013

全域變數 hexo 本身是一個 EventEmitter 物件，你可直接監聽或觸發事件。

使用

監聽事件： 

hexo.on('ready', function(){
  ...
});

觸發事件： 

hexo.emit('event', [arg1], [arg2], [….]);

參考

預設事件

ready

設定和所有外掛都載入完畢後。

generateBefore

開始生成檔案前。

generateAfter

生成檔案完成後。

new

建立新文章後。回傳一個參數 target，代表新檔案的絕對路徑。

server

伺服器開啟後。

preview

預覽伺服器開啟後。
