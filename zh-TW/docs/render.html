渲染

Last updated: Mar 2, 2013

使用

render(data, [options], callback)

  • data - 輸入資料
    • path - 路徑
    • text - 字串
    • engine - 渲染引擎（預設為path的副檔名，path未定義則此為必填項目）
  • options - 選項
  • callback - 回呼函數

renderSync(data, [options])

同步模式的 render.

renderFile(path, [options], callback)

  • path - 路徑
  • options - 選項
    • cache - 快取
    • layout - 布局
  • callback - 回呼函數

範例

hexo.render.render({path: layout.ejs}, function(err, content){
  // ...
});
