外掛

使用

安裝

執行以下指令。 

npm install <plugin-name> --save

啟用

在_config.yml的plugins欄位中加入外掛名稱。 

plugins:
- plugin-one
- plugin-name

停用

從_config.yml的plugins欄位中移除外掛名稱。 

plugins:
- plugin-one

更新

執行以下指令。 

npm update

移除

執行以下指令，在移除前別忘了先停用外掛。 

npm uninstall <plugin-name>
